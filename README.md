# FileCC

**FileCC** is a cross-platform desktop application for fast and reliable document and image conversion.

## ✅ Features
- Convert DOCX, PDF, TXT, HTML files to TXT, HTML, or PDF.
- Convert images: JPG, PNG, WebP, AVIF, TIFF, BMP, GIF.
- Electron-based installer for Windows x64.
- Optimized file handling for large documents and images.
- Improved PDF text extraction and text-to-PDF rendering.
- Bug fixes and performance improvements.

## ⚡ Quick Start
1. Download and install **FileCC** from the [Releases](https://github.com/Bloby22/FileCC/releases) page.
2. Open the app.
3. Drag & drop your files or use the built-in file selector.
4. Choose your output format and click **Convert**.
5. Enjoy simple, fast, and accurate conversions.

## 🖥️ Supported Platforms
- Windows x64 (installer included)
- Other platforms coming soon
